# 09_storage_impl

이 폴더의 목표는 **CSV 파일을 실제로 읽고 쓰는 코드**를 만드는 것입니다.

이제 파서와 실행기가 준비되었으니,
실제로 `03_data/materials.csv` 파일을 다루는 구현이 필요합니다.

## 이 단계에서 만드는 파일

- `storage.c`

## 이 파일이 하는 일

- `append_row_to_table()`:
  값을 `LX2 sunvisor,ASD,MMH,21G` 같은 CSV 한 줄로 만들어 파일 끝에 추가합니다.
- `print_table_rows()`:
  파일 내용을 읽어서 그대로 화면에 출력합니다.
