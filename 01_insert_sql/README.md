# 01_insert_sql

이 폴더의 목표는 **코드를 만들기 전에 입력 형태를 먼저 눈에 익히는 것**입니다.

우리는 나중에 C 코드로 SQL 문자열을 읽고 해석할 예정입니다.
하지만 초보 단계에서는 먼저 "프로그램이 어떤 입력을 받는가"를 분명히 알아야 합니다.

## 이 단계에서 만드는 파일

- `insert_user.sql`
- `insert_material.sql`

## 이 SQL이 의미하는 것

```sql
INSERT INTO users VALUES ('Alice', '20');
```

재료 예제는 아래처럼 바로 넣을 수 있습니다.

```sql
INSERT INTO materials VALUES ('LX2 선바이저', 'ASD', 'MMH', '21G');
```

위 문장은 다음 뜻입니다.

- `INSERT`: 새 데이터를 넣어라.
- `INTO users`: `users` 테이블에 넣어라.
- `VALUES (...)`: 실제 넣을 값은 이것들이다.
- `;`: SQL 문장이 여기서 끝난다는 표시다.
